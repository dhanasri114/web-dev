Hyperlink-Tag

The hyperlink tag are used to link one page to another page

< a > tag define a hyperlink

hyperlink has two attibute:

  1. href:- the href attribute define the url where the link point
    • Example: Visit google

  2. target:- the target="_blank" attribute are used to linked page open in new tab
    • Example: visit google in new tab
