List Tag

there are two types of list:

1.ordered list

2.unordered list

ordered List

  1. ordered list has two attributes are

    1. start:- to specific number
    2. type:- like numbers,alphabate,romannumbers.

  2. Example:
    1. mango
    2. banana
    3. apple

un-ordered List

  • un-ordered list has only one attribute are
    • type:- like circle,squre

  • Example:
    • mango,a
    • banana
    • apple

Note:

  • In list tag has only one attibute is value
