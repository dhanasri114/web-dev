CSS - Seletor

ID-Selector

  • ID seletor use '#' symbol
  • Example: < h1 id="id_name" >
    #id_name{
    write styles
    }

change background color use id selector

Class-Selector

  • Class seletor use '.' symbol
  • Example: < h1 class="class_name" >
    .class_name{
    write styles
    }

change background color use class selector

Type-Selector

  • Type seletor use Tag namss
  • Example:tag_name{
    write styles
    }

this is type selector
